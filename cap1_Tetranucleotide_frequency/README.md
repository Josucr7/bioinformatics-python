# Tetranucleotide Frequency

## Description
This project calculates the frequency of each nucleotide (A, C, G, T) in a DNA sequence provided as a text input or file.

## Concepts Practiced
- Using 'argparse` to handle command-line arguments
- Accepting input from standard input or files
- Counting and reporting nucleotide frequencies
- Basic string processing in Python

## Requirements
- Python 3
- No external libraries required

## Usage
```bash
python DNA.py input_file
