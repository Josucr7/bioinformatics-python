# Exploring Sequence Similarity

## Description
This project searches for the occurrences of a DNA subsequence within a DNA sequence and reports the positions where it is found.
The program detects overlapping occurrences of the subsequence.

## Concepts Practiced
- Using argparse to handle command-line arguments.
- Matching subsequences within DNA sequences.
- Working with sequence indexing and pattern searching.

## Requirements
- Python
- DNA sequence
- DNA subsequence

## Usage

```bash
python subseq_dna.py "dna_sequence" "sub_sequence"
```
