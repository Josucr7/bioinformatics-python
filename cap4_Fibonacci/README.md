# Generation of the Last Number of a Fibonacci Sequence

## Description
This project generates the last number of a Fibonacci sequence based on the number of generations and litter size provided by the user.

## Concepts Practiced
- Using argparse to handle command-line arguments.
- Working with generators in Python.
- Applying recursion and sequence generation concepts.
- Using NamedTuple for structured arguments.

## Requirements
- Python 3

## Usage
```bash
python fibonacci.py gen lit
```

Example:

```bash
python fibonacci.py 5 3
```
