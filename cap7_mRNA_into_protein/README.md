# Translate mRNA Sequences into Protein Sequences

## Description
This project demonstrates two different approaches for translating an mRNA sequence into a protein sequence.

The first approach uses custom Python logic and a codon dictionary. The second approach uses the Biopython library to perform the translation.

## Objectives
- Apply multiple methods to solve the same problem.
- Use biological concepts such as start and stop codons.
- Handle different input formats, including raw sequences and FASTA files.
- Compare a manual implementation with a Biopython-based solution.
