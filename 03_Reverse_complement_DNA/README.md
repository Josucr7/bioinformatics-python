# Generation of the Reverse Complement Sequence of a DNA sequence.

## Description
This project generates the reverse complement sequence of a DNA sequence using the Biopython library.

## Concepts Practiced
- Using argparse to handle command-line arguments.
- Reading string inputs as DNA sequences.
- Applying the Biopython library.

## Requirements
- Python 3
- Biopython
- Input DNA sequences

## Usage
```bash
python reverse.py DNA
```
