# Translate an mRNA Sequence into Protein Sequence using basic Python Logic.
## Description
This project translates an mRNA sequence provided as a string. The program identifies codons and uses a dictionary to select the corresponding amino acids.
It also analyzes for the presence of a start codon, if the condition achieve the beginning the translation process and stops when a stop codon is encountered.
## Concepts Practiced
- Using argparse to handle command-line arguments.
- Translating mRNA sequences into protein sequences.
- Applying biological concepts such as stop and start codons.
- Using dictionaries to map codons to amino acids.

## Requirements
- Python
- mRNA sequence
## Usage
```bash
python mRNA_protein.py "mRNA sequence"
```
